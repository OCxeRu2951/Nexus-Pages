Privacy Policy

Last updated: May 14, 2026

OCxeRu2951（以下「開発者」）は、Discord Bot「Nexus」（以下「本Bot」）の利用に際して収集する情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

収集する情報

本Botは以下の情報を収集・保存します。

ユーザー情報
Discord ユーザーID・ユーザー名
サーバー情報
ギルドID・チャンネルID・ロールID
コマンドデータ
投票・警告・申請・AFK等のデータ
設定情報
各サーバーの設定値

収集しない情報

  • メッセージの内容（コマンド実行時のコンテンツを除く）
  • パスワード・認証情報
  • メールアドレス・電話番号等の個人識別情報
  • 位置情報

情報の利用目的

  • 本Botの機能提供（投票・モデレーション・時報等）
  • サービスの改善・デバッグ
  • 不正利用の防止

情報の保存期間

  • AFK・投票データ: サーバー設定に準じる（デフォルト7〜90日）
  • 警告データ: デフォルト90日
  • 申請データ: デフォルト90日
  • モデレーションログ: サーバーが削除するまで
  • サーバー設定: Botがサーバーを退出するまで

第三者への提供

収集した情報を第三者に販売・提供することはありません。ただし、以下の場合を除きます。

  • 法律・法的手続きにより開示が求められた場合
  • サービス提供に必要なインフラ事業者（Turso・Cloudflare・GCP）への必要最低限の情報共有

データの削除

以下の方法でデータを削除できます。

  • サーバーからBotを除外する（そのサーバーの全データが削除されます）
  • GitHubのIssueまたはサポートサーバーにて削除を申請する

セキュリティ

データはTursoデータベースに暗号化して保存されます。開発者以外がデータに直接アクセスすることはありません。

ポリシーの変更

本ポリシーは予告なく変更される場合があります。重大な変更がある場合はサポートサーバーにてお知らせします。

お問い合わせ

プライバシーに関するお問い合わせは、GitHubのIssueまたはサポートサーバーよりご連絡ください。

This Privacy Policy describes how OCxeRu2951 ("Developer") collects, uses, and handles information when you use the Discord Bot "Nexus".

Information We Collect

User Data
Discord user IDs and usernames
Server Data
Guild IDs, channel IDs, role IDs
Command Data
Polls, warnings, applications, AFK data
Settings
Per-server configuration values

Information We Do Not Collect

  • Message contents (except command input)
  • Passwords or authentication credentials
  • Email addresses or phone numbers
  • Location data

How We Use Information

  • To provide Bot features (polls, moderation, announcements, etc.)
  • To improve and debug the service
  • To prevent abuse

Data Retention

  • AFK / poll data: Per server settings (default 7–90 days)
  • Warning data: Default 90 days
  • Application data: Default 90 days
  • Moderation logs: Until deleted by the server
  • Server settings: Until the Bot leaves the server

Third Parties

We do not sell or share your data with third parties, except:

  • When required by law or legal proceedings
  • Minimal sharing with infrastructure providers (Turso, Cloudflare, GCP) necessary for service operation

Data Deletion

  • Remove the Bot from your server (all server data will be deleted)
  • Submit a deletion request via GitHub Issues or the support server

Security

Data is stored encrypted in Turso database. Only the Developer has direct access to the data.

Policy Changes

This policy may be updated without notice. Significant changes will be announced in the support server.

Contact

For privacy inquiries, please contact us via GitHub Issues or the support server.
